Stuart Wurtman

I study math and computer science at CSU Bakersfield. Some of my favorite studies have been in the subjects of algorithms, software engineering, and computer architecture. I’ve had the opportunity to develop a video game using the Unity Platform, develop an Enterprise Resource Planning system using a database, and solve large math problems using computation. My experience has taught me that projects are the best way to learn – learn by doing. After finishing my undergraduate education, I want to keep learning, either in a graduate program or in industry. Computers allow us to take a small set of input rules or data and create a massive, complex output. I know that working with purpose, I can use computers to make a difference. I have a keen interest in the outdoors, and I hope my work can help protect our environment and ensure equal access to the outdoors for everyone. Environmental justice is social justice.

Bike Shop RMS

As part of a group, I am developing a prototype retail management system (RMS) tailored to a bike shop. The purpose of this project is to study the implementation of a database and database management software.

Project Euler

Project Euler is a set of challenging math problems that require computation to solve. I am currently working through these problems as a means to learn efficient algorithm design in python.

  • Learn more

Tidal Prediction Algorithm

Modern tidal prediction is based on harmonic analysis. Tidal predictions are made by combining usually between 20 and 70 constituent curves. I am learning the math behind tidal predictions and implementing an algorithm that can predict tides accurately, while remaining more space efficient than storing raw tidal prediction data, and time-efficient enough to run on wearable devices.
